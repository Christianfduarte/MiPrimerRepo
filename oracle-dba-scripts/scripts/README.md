# Índice de categorías de scripts

Cada subcarpeta representa un dominio técnico Oracle DBA.

- `dataguard`: sincronización, transporte y apply lag.
- `rac`: estado de instancias y servicios en cluster.
- `exadata`: métricas y salud específica de plataforma Exadata.
- `rman`: estado de jobs y piezas de backup.
- `performance`: top SQL, waits y recursos.
- `asm`: uso y estado de disk groups ASM.
- `multitenant`: estado de PDB/CDB.
- `monitoreo`: sesiones activas, bloqueos y alertas.
- `seguridad`: roles y privilegios.
- `usuarios`: cuentas, estado y perfiles.
- `mantenimiento`, `storage`, `auditoria`, `networking`, `parcheo`: operación y gobierno.
