# Índice de categorías de scripts

Cada subcarpeta representa un dominio técnico Oracle DBA.

- `dataguard`: sincronización, transporte, apply lag y procesos managed standby.
- `rac`: instancias, servicios y eventos global cache.
- `exadata`: offload/smart scan y métricas específicas.
- `rman`: jobs, resumen semanal y archivelogs sin backup.
- `performance`: top SQL por CPU/elapsed, waits y consumo TEMP.
- `asm`: uso y estado de disk groups/discos ASM.
- `multitenant`: estado de PDBs y save state.
- `monitoreo`: sesiones activas, bloqueos y longops.
- `seguridad`: roles, privilegios críticos y grants sensibles.
- `usuarios`: cuentas, perfiles y actividad.
- `mantenimiento`: objetos inválidos y jobs con error.
- `storage`: umbrales críticos, datafiles y autoextend.
- `auditoria`: logins y errores de autenticación.
- `networking`: sesiones por origen y servicios activos.
- `parcheo`: inventario SQL patch y estado de componentes.
